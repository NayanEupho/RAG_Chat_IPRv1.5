# Example: Technical FAQ (Question Label Format)
## Project Eupho
### API Integration

Question: How do I authenticate with the REST API?
Answer: You must include a `Bearer` token in the `Authorization` header. Tokens can be generated via the Developer Dashboard under the 'API Keys' section.

Question: What is the rate limit for the free tier?
Answer: The free tier is limited to 1,000 requests per day per IP address. Exceeding this will result in a 429 status code.
